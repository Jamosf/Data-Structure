## 动态规划
动态规划给我们提供了一个解决问题的很好地思路，这个思想极大地帮助了我分析问题。在没有接触动态规划之前，遇到这类问题，最直觉的想法就是暴力匹配，暴力匹配解决问题本身没有什么问题，因为暴力匹配可以解决大部分小样本的情况。可是，每一题都采用这样的方法，难免让人烦恼，因为缺少了思考问题的乐趣。学习了一个新的知识，才会发现知识的力量是多么伟大，从而更加激励自己去学习更多的东西。勉！
